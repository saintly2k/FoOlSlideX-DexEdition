Naming Rules

Order Number
The absolute order. Sorts all chapters by this number. If left blank, the chapter number will be inserted.
Chapter Number
Number only, decimals allowed. Example: 0 1 3.5 18
Use 0 for Prologue and name the release name Prologue below.
Use decimals (Example: 1.5) for special/bonus/omake/etc.
Use 1 for Oneshot and name the release name Oneshot below.
Volume Number
Number only, decimals not allowed. Example: 1 2 3 18
Release name
By default, we will automatically generate a release name based on the value of volume number and chapter number.
You can customize the release name by filling in this field. However, automatic sorting is still determined by the values of chapter and volume, and is not affected by the release name you inputed here.
Example: Chapter 6 Episode 99 Season 2 Ep 18 ISSUE 23 Punch 1024 Day 1001 Prologue Oneshot
Release name short Version
Short version of the release name. May be used on some small screen devices.
Example: C6 E99 S2E18

Archive Order

Only .zip archives are allowed! They cannot have subdirectories and cannot be password protected. The webserver displays all images ordered by their name. For example, there are images 1.png 2.png 3.png, it will display them as 1 2 3. However, if there are images like 1.png 5.png 8.png 10.png, it will order them like 1 10 5 8. To prevent this, please zero-pad the images. (01.png, 05.png, 08.png, 10.png) to achieve the proper order.
